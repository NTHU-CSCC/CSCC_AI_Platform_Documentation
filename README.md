# CSCC AI Platform Documentation

User documentation for the **GPU Cluster Management Platform** operated by NTHU CSCC — covering getting started, workspaces, projects, groups, storage, and settings.

🌐 **Live site:** https://nthu-cscc.github.io/CSCC_AI_Platform_Documentation/

## Languages

The site is published in two languages, with Traditional Chinese as the default:

- **繁體中文 (default):** [`guides-zh-tw/`](guides-zh-tw/)
- **English:** [`user-guide/`](user-guide/)

## Contents

- **User Guide** — Getting Started, Concepts & Roles, Home Dashboard, Workspaces, Projects, Groups, Storage, Requests, Settings, CLI, Announcements
- **Admin Guide** (Chinese) — User Management, Group Management, Project Management, Storage Management, Resource Plans

## About this repository

This repository hosts the **built static site** served by GitHub Pages. The Markdown sources and MkDocs configuration are maintained separately in the `docs-builder` project and published here on each release.

- Built with [MkDocs](https://www.mkdocs.org/) + [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
- Application repository: [NTHU-CSCC/CSCC_AI_Platform](https://github.com/NTHU-CSCC/CSCC_AI_Platform)
